RESTAURANT PHOTOS — how the site loads them
============================================

The site now uses the photos in THIS folder, named by restaurant, e.g.:
    Carmelina_s-1.png, Carmelina_s-2.png, ...
    Gordon Ramsey-1.png ...
    Tatte Bakery & Cafe-1.png ...
Up to 5 photos per restaurant are shown (slots 1-5). Add more or fewer freely;
restaurants with NO photos here simply show no gallery (just Map / Google / Yelp buttons).

Naming: use the restaurant name + "-" + number. Spaces, apostrophes, & and accents
are all fine. (The site matches on letters & digits only, so "Carmelina_s" and
"Carmelina's" both work.)
