BACKGROUND PHOTO — HOW TO USE YOUR WOLLASTON BEACH PICTURE
==========================================================

The itinerary site looks for your beach photo here:

    images/wollaston-beach.jpg

To use the photo you uploaded:
1. Save / rename your Wollaston Beach photo as:  wollaston-beach.jpg
2. Put it in this "images" folder (next to Boston-2026-Itinerary.html).
3. When you upload the site to your web server, upload this whole "images"
   folder alongside the HTML file.

That's it — the page will automatically use it as the background.
If the file isn't there, the site falls back to a beach-colored gradient,
so it never looks broken.

(JPG, PNG, or WEBP all work — just keep the name "wollaston-beach.jpg",
or update the first entry in the "bg" list inside the HTML.)
